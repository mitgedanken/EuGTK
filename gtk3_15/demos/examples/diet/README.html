EuGTK Sample Listview

Warning! This only works on Linux.


The new listview cell formatting options include blank-when-zero, formatting using text:format() patterns, fancy column colors, and decorative column headers.

DIET.ex source code is well-commented so that you can see how to use these features in your own code.
