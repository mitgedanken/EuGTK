Euphoria/EuGTK Syntax for Geany

The Geany editor is available for Windows, Linux and MacOSX.

The files here allow you to do syntax highlighting for the Euphoria programming language.

I have included a modified version of Kenneth Rhodes' gneui.ex program which allows Geany to jump to the location of an error. To use it, just set your Geany Build Command in the Execute box to: eui ~/demos/gneui.ex %f  and move gneui.ex to your working directory (e.g: ~/demos). If there are problems with this, it's due to my modifications, blame not the original author!

In the ~/demos/resources/geany_syntax folder you will find two config files to trick Geany, Pluma, etc. into thinking your Euphoria files are Lua source code, thus giving you syntax highlighting: 
Copy filetype_extensions.conf into ~/.config/geany/
Copy filetypes.lua into ~/.config/geany/filedefs

When you open an .ex file, use the Geany menu option:
Document/Set Filetype/Scripting Languages/Lua source file
to enable color syntax coding.

After restarting Geany, Geany's Execute button will call eui, the Build button runs eubind, and the Compile button starts euc (as a background task).
