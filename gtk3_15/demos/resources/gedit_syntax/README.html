Euphoria/EuGTK Syntax for text editors

These files allow you to add syntax highlighting for the Euphoria programming language to several of the most common cross-platform editors, including the GtkSourceView editor used by the BEAR.

To Install:

For programs using GTK2 -- i.e. Pluma, gedit, etc.

sudo cp euphoria.lang /usr/share/gtksourceview-2.0/language-specs/euphoria.lang
sudo cp gedit_syntax.xml /usr/share/mime/packages/
For programs using GTK3 -- i.e. Xed, EuGTK Sourceview
sudo cp euphoria.lang /usr/share/gtksourceview-3.0/language-specs/euphoria.lang
sudo cp gedit_syntax.xml /usr/share/mime/packages/

Next, copy all the styles included here in "one swell foop":

sudo cp *.xml /usr/share/gtksourceview-2.0/styles/
sudo cp *.xml /usr/share/gtksourceview-3.0/styles/
You may have to change permissions on the following files:
sudo chmod 777 /usr/share/gtksourceview-2.0/language-specs/euphoria.lang
sudo chmod 777 /usr/share/gtksourceview-3.0/language-specs/euphoria.lang

For more themes, go to gedit-themes on github.
